# 📁 Template Images - Folder Structure Guide

## Quick Start

Upload your template images to the appropriate folders:

```
public/resume-examples/
├── aurora/preview.png         ← Upload Aurora template image here
├── hyperion/preview.png       ← Upload Hyperion template image here
├── lunar/preview.png          ← Upload Lunar template image here
├── stellar/preview.png        ← Upload Stellar template image here
├── zenith/preview.png         ← Upload Zenith template image here
├── aether/preview.png         ← Upload Aether template image here
├── nebula/preview.png         ← Upload Nebula template image here
├── eon/preview.png            ← Upload Eon template image here
├── cosmos/preview.png         ← Upload Cosmos template image here
├── modern/preview.png         ← Upload Modern template image here ⭐ Bestseller
├── classic/preview.png        ← Upload Classic template image here
└── creative/preview.png       ← Upload Creative template image here
```

## Image Specifications

- **Filename**: Must be `preview.png` or `preview.jpg`
- **Dimensions**: 1700 x 2200 pixels (8.5:11 ratio)
- **Format**: PNG (recommended) or JPG
- **Size**: < 500KB (optimized for web)

## Folder Purpose

Each template folder can contain:
1. `preview.png` - **Required** - Main image shown on templates page
2. `example-1.png` - Optional - Additional variations
3. `example-2.png` - Optional - More examples
4. `README.md` - Template documentation
