## Color Scheme

### Premium Badges:
- **Premium (Active)**: `bg-emerald-100 text-emerald-700` (green)
- **Lock (Inactive)**: `text-amber-600` (amber)

### Plan Colors:
- **Weekly**: Blue gradient (`from-blue-500 to-cyan-500`)
- **Monthly**: Purple gradient (`from-purple-500 to-pink-500`)
- **Annual**: Gold gradient (`from-yellow-500 to-orange-500`)

### Button States:
- **Fix Spelling**: Green (`bg-white hover:bg-green-50 border-green-300 text-green-600`)
- **Fix Grammar**: Purple (`bg-white hover:bg-purple-50 border-purple-300 text-purple-600`)
- **Enhance AI**: Blue (`bg-white hover:bg-blue-50 border-blue-300 text-blue-600`)

### AI Features Box:
- **Border**: `border-2 border-purple-200`
- **Background**: `bg-gradient-to-br from-purple-50/50 to-blue-50/50`
- **Header Text**: `text-purple-900`
- **Icon**: `text-purple-600`

## Responsive Behavior

### Desktop (≥768px):
- Subscribe page: 3 columns (all cards side-by-side)
- Payment page: 2 columns (form | summary)
- AI features: All 3 buttons in a row

### Mobile (<768px):
- Subscribe page: 1 column (cards stack)
- Payment page: 1 column (summary below form)
- AI features: All 3 buttons in a row (may wrap)

## Animation & Interactions

### Subscribe Page:
- Cards: `hover:scale-105` (slight zoom on hover)
- Selected card: `scale-105 ring-4 ring-blue-100` (highlighted)
- Popular badge: Gradient background with pulse animation

### Payment Page:
- Form fields: Focus states with blue border
- Submit button: Gradient with hover darkening
- Processing: Spinner animation
- Success: Fade in animation

### AI Features Box:
- Buttons: Hover color transitions
- Disabled state: `opacity-50` with subtle cursor
- Loading state: Spinner in button

## Accessibility Features

### Keyboard Navigation:
- All buttons focusable
- Tab order follows visual flow
- Enter key submits forms
- Escape closes dialogs

### Screen Readers:
- Semantic HTML (`<header>`, `<main>`, `<section>`)
- ARIA labels on icon buttons
- Form labels properly associated
- Error messages announced

### Visual Indicators:
- Color is not the only indicator
- Icons supplement color
- Text labels on all actions
- Loading states clearly communicated
