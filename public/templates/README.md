# Template Images Folder

This folder contains high-resolution images of resume templates for the templates selection page.

## Image Requirements

- **Format**: PNG or JPG
- **Aspect Ratio**: 8.5:11 (standard US Letter paper size)
- **Recommended Resolution**: 1700 x 2200 pixels (for high-quality display)
- **File Size**: Optimized for web (aim for under 500KB per image)
